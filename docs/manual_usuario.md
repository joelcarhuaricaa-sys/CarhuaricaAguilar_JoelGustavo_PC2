# Manual de Usuario y Guía de Ejecución

## Requisitos
- Compilador C++ compatible con el estándar **C++17**.
- **CMake** versión 3.10 o superior.

## Instrucciones de Compilación
Desde la raíz del proyecto (`CARHUARICAAGUILAR_JOELGUSTAVO_PC2`), ejecuta los siguientes comandos en tu terminal:

### Compilación
```bash
cmake -S . -B build
cmake --build build
```

### Ejecución
```bash
./build/tu_programa
```
